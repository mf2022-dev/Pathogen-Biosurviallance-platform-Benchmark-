# BioR — Biosurveillance Intelligence for Operational Readiness

**Built on PSEF v3.0.0 Canonical Baseline: 169 Platforms | 14 Dimensions | 1,690+ Data Points**

BioR is a strategic intelligence product for global biodefense posture assessment. It answers three fundamental questions: **What exists?** (Global Threat Landscape Map), **What is missing?** (Capability Gap Analysis), and **What should connect?** (Interoperability Assessment).

## Canonical Reference Baseline

The `reference_baseline/` directory contains the frozen authoritative dataset:

- `CANONICAL_169_PLATFORMS.json` — Full structured data (all scores, dimensions, strengths, weaknesses)
- `CANONICAL_169_PLATFORMS.csv` — Flat tabular export
- `REFERENCE_BASELINE.md` — Manifest with complete 169-platform registry and usage rules

**Rules:** Never modify files in `reference_baseline/`. Updates require a new version (e.g., `CANONICAL_v3.1_PLATFORMS.json`).
