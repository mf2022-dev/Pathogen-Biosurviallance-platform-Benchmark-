# AI Developer Instructions

## Project: Pathogen Surveillance Evaluation Framework v3.0.0

### Quick Reference
- **Repository:** pathogen-surveillance-evaluation
- **Branch:** main
- **Deployment:** GitHub Pages (from main branch, root)
- **Total Files:** 35 key files across 8 directories

### For AI Agents
1. All HTML files are self-contained with CDN dependencies
2. No build process required - static HTML/CSS/JS
3. Data files in `data/` directory contain all evaluation data as JSON
4. GitHub Pages deployment from root of main branch
5. Jekyll config in `_config.yml` handles GitHub Pages settings

### Key Data Files
- `data/platform_evaluation_data.json` - All 56 platform evaluations
- `data/evaluation_criteria_detailed.json` - Scoring criteria and weights

### Deployment Steps
1. Push to GitHub repository
2. Enable GitHub Pages: Settings → Pages → main branch / root
3. Site available at: `https://username.github.io/repo-name`
