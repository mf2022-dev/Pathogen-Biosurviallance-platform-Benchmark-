# Contributing to PSEF

## How to Contribute

1. Fork the repository
2. Create a feature branch
3. Add or update platform evaluations
4. Submit a pull request

## Evaluation Guidelines

- Use the evaluation template in `evaluation_tools/`
- Score each of the 10 dimensions from 0-100
- Provide evidence-based justification for scores
- Include strengths, weaknesses, and use cases
- Reference the scoring rubric for consistency

Please follow our [Code of Conduct](CODE_OF_CONDUCT.md).
