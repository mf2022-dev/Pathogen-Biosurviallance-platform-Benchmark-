# Quick Start for AI Developer

## Deploy in 3 Steps

1. **Push to GitHub:**
   ```bash
   git remote add origin https://github.com/mf2022-dev/Pathogen-Biosurviallance-platform-Benchmark-.git
   git push -u origin main
   ```

2. **Enable GitHub Pages:**
   - Repository Settings -> Pages -> Source: main branch / root

3. **Verify:**
   - Visit `https://mf2022-dev.github.io/Pathogen-Biosurviallance-platform-Benchmark-/`
   - Check all navigation links work
   - Verify JSON data loads correctly
